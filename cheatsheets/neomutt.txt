**Cheatsheet of less obvious commands:**
```
COMPOSING:
m       - New message
r       - Reply
g       - Group reply (reply-all)
f       - Forward
y       - Send message (after composing)
e       - Edit message again (before sending)

NAVIGATION:
j/k     - Down/up (vim-style)
/       - Search
n       - Next search result
c       - Change folder
?       - Show help

MANAGING:
d       - Mark for deletion
u       - Unmark (undelete)
$       - Sync/purge (actually delete marked messages)
s       - Save message to folder
t       - Tag message (for bulk operations)
T       - Tag pattern
;d      - Delete all tagged messages
ctrl+b  - Refresh mailbox
$       - Refresh mailbox

READING:
Enter   - Open message
i       - Back to index
v       - View attachments
|       - Pipe message to command

OTHER:
q       - Quit
x       - Quit without saving changes
!       - Go to first folder (INBOX)
